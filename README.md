Curtly is a multimodal study assistant designed for the modern student. Moving beyond simple text extraction, Curtly v2.0 focuses on Information Synthesis. It bridges the gap between complex academic sources and student comprehension by expounding on files and videos with real-world context.

Developed by Kurt, a BS Psychology student at USL, Curtly was born from the need for a tool that understands Taglish, simplifies heavy jargon, and handles everything from PDFs to YouTube lectures.
